== Description ==

Creates a widget to display your App.net status.  Also serves as a sample application for accessing and updating data in App.net.

== Installation ==

1. Upload all files to the `/wp-content/plugins/adn-profile` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the ADN Status widget to your sidebar under Appearance=>Widgets

== Screenshots ==
